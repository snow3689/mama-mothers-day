# 致 妈妈 · 母亲节快乐

一个写给妈妈的单页交互网站。

> 你从不喊苦，也从不抱怨。<br/>
> 所有的难，你都默默地放在自己身后；<br/>
> 所有的好，你都悄悄地留给了我和妹妹。

## 在线访问

部署在 GitHub Pages，手机直接打开链接即可。

## 本地预览

直接双击 `index.html` 在浏览器里打开。

## 结构

```
index.html        — 全部页面 + 样式 + 交互
photos/           — 写真 / 全家福 / 爸妈合照
```

## 技术

- 原生 HTML / CSS / JS，单文件
- Web Audio API 现场合成环境音乐（C – Am – F – G 暖色和声进行）
- IntersectionObserver 滚动入场动画
- 自定义光标 / 飘落花瓣 / 灯箱 / 触屏滑动
- iOS safe-area-inset 适配
